# 🧠 Gerador de Portfólio Profissional (ou quase isso 😏)

Se você já tentou montar um portfólio e pensou:

> "isso aqui podia ser automático..."

Parabéns. Você pensou exatamente como um engenheiro de verdade.

---

## 🚀 O que esse projeto faz

Este humilde (porém poderoso) sistema:

- 📝 Permite preencher **todos os dados do seu portfólio direto na tela**
- 📸 Faz upload da sua foto (sim, só JPEG — aqui tem padrão)
- 🎯 Organiza tudo em **seções profissionais**
- 📄 Gera um **PDF formatado automaticamente**
- 🧱 Estrutura modular (sim, alguém aqui sabe arquitetura)

Basicamente:

> Você digita → ele organiza → você baixa → LinkedIn fica com inveja

---

## 🧩 Módulos do sistema

O sistema foi pensado como um mini-produto:

- 📌 **Contato**
- 🧠 **Habilidades & Especialidades**
- 🛠 **Tecnologias**
- 🧾 **Resumo profissional**
- 💼 **Portfólio**
- 🏢 **Experiência**
- 🎓 **Cursos**

Tudo editável. Tudo exportável. Tudo sob controle.

---

## 🧪 Tecnologias utilizadas

- HTML (raiz)
- CSS (sem frescura, direto ao ponto)
- JavaScript (fazendo o trabalho pesado)
- jsPDF (o cara que transforma tudo em PDF bonitinho)
